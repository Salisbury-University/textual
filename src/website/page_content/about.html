About

Traditionally, collections of English language data have been inaccessible, limited to a specific time period, or expensive. This project is attempting to remedy all of these issues at once with the creation of an Open Source English Textual Baseline. The data available has been gathered from a variety of open source websites and collections. The dataset may be freely downloaded on browsed from this site.

Currently the dataset contains ~320 million unique entries and is continuing to grow. Using this collection of data as a baseline will allow for inference on questions pertaining to how humans use and have used the English Language. Some examples of possible avenues of research can be explored in our tutorials section.

Sources

Reddit

The Reddit API is a powerful tool for collecting English textual data. The API provides unlimited access to Reddit content, allowing for the stockpiling of very large amounts of data. In addition to being easily accessible, the content on Reddit is usually non-formal, giving a better representation of how the English language is actually used.

Reddit »

Project Gutenberg

Project Gutenberg is a large collection of open source eBooks. It contains sources accross all genres and all historical significance. It is the oldest digital library, founded in 1971, and contains over 60,000 eBooks. Since the books span across many years and genres, the books can be used for textual analysis to see how language has changed over time.

Project Gutenberg »

Yelp

The Yelp review site provides access to a large collection of reviews across a large variety of categories. In addition to containing the review text, the Yelp review dataset has the number of stars specified in each review. These review rating can be used for aditional analysis, such as training a neural network predict the number of stars a review should have based on its wording.

Yelp »

Amazon

The collection of Amazon reviews provide insights to user's feelings on products dating back to as early as 1997. This is a great source in order to undersand the true opinions people felt about the products that they are buying. Amazon has an international reach as well, so you are able to get a whollistic view.

Amazon »

Twitter

Twitter gives us another point of analysis for people's social media behavior on certain issues, events, and just conversing with one another over the internet. The Twitter developer API allows us collect tweets from specific users like news organizations, political figures, and other influencers and recent random tweets of a specific query given. Allows for a great analysis on communication over social media.

Twitter »

WikiSource

Similar to Project Gutenberg, the WikiSource Online Digital Library contains a massive collection of open source English works. Ranging from legal documents to general literature the site provides access to many different sources of textual data. Currently, the site contains over 560,000 English works with many more in other languages. In combination with the Project Gutenberg source, the WikiSource site will be used to provide a more formal set of textual data.

WikiSource »

YouTube

YouTube comments provide a way for users to share their thoughts about a video or its topic. The Youtube API provides an access point to retrieve comments from videos of every category. The collection of YouTube comments can be used for textual analysis such as sentiment analysis or topic modeling.

YouTube »

Google Reviews

Google Reviews is a comprehensive platform where users provide a wide range of reviews across various categories. Each review includes a star rating, allowing for additional analysis like training neural networks to predict ratings based on the review's text. This dataset offers valuable insights to businesses and users, enhancing their understanding of the reviews and ratings they encounter.

Google »

ABC Australia

The ABC Australia dataset is a diverse collection of information gathered by the Australian Broadcasting Corporation (ABC). It includes news articles, videos, audio recordings, and other media content, providing researchers with valuable Australian-centric data for in-depth analysis and exploration. With its comprehensive coverage, the ABC Australia dataset serves as a valuable resource for studying Australian news, media, and culture.

ABC Australia »
