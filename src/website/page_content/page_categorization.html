Latent Dirichlet Allocation

What is Latent Dirichlet Allocation?

A crucial portion of understanding what a document is truly about stems from looking at its most relevant words. Latent Dirichlet Allocation is a way to identify topic words. This type of topic modeling is a method of unsupervised classification of documents, which allows for easier organization, searching, and summarizing archives such as the one we have here.

Ideally, each document in each collection in our database will soon receive their top words, which will allow easy querying of the database in order to find documents related to a certain topic. This will facilitate the creation of visualizations based off of the web of related topic words that will span from multiple documents.

This program uses the gensim library in Python, which contains the tools for creating an LDA model. It has the LDAMulticore (a parallelized version of an LDA) object that was trained on each of the collections of data. There will be an LDA for each of the collections in order to better the accuracy of the models, as the breadth of information present in each collection varies widely.
