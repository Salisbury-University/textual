Our Sample Work

Review Classification

Reviews are often a showcase of how people really feel. The Yelp review site provides access to a large collection of reviews from various different categories ranging from restaurants to airlines. What can we learn from these reviews? This example showcases one possible area of research by attempting to classify reviews on a 5-star scale using Machine Learning.

Text Sentiment

Sentiment analysis of the textual content can help futher classify data into categories. Using the Natural Language Tool Kit (NLTK) and SpaCy blocks of text are taken and analyized. Each piece of data is then classified into one of three categories, positive, neutral, or negative. By using this process we can categorize different time periods of the data to see if peoples sentiment changes over time.

Page Categorization

Page categorization helps understand what the content of each of the data collections is. Initially designed as a neural network, this now functions utilizing a tool called Latent Dirichlet Allocation. Topic modeling is a method for unsupervised classification of documents which is crucial for our project. This utilizes NLTK and gensim for the LDA classifier.
