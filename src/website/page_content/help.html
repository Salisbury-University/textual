Website Help

This page serves as place to hold comprehensive documentation related to this project. Users seeking information about specific parts of this website, dataset, or source code should consult this page for more information. In addition to the tutorial located under the "tutorial" tab, this page seeks to provide a baseline to get users started with this dataset. Categories for different help sections can be found below.

The topics covered will include the outline of this website, basic website functions, an example of downloading a subset of the dataset, and a breif overview of how to load this data into Python for further processing. More in depth examples of analysis showing what can be done using this dataset can be found under the tutorials page. Examples of the content contained in this database are located under the visualizations page.

Website Structure

Home: This page serves as the source for all the pages on this site. From here users are able to navigate to the various different website functions and documentation sections.

About Project: The about page hosts information about the structure of this project and how the various web scraping tools operate to collect the data. Additionally, brief explainations of the major sources pulled from are provided alongside links to the original pages.

Meet the Team: A brief explaination of each of the members of our team. Each member has a paragraph with a small information block about them. Additionally, a portrait and a link to each member's GitHub is displayed.

Downloads: Provides users with access to the database. The page displays the contents of a specified collection. Additionally, users are able to select which collection they would like to download data from with the depth of the sample able to be entered based on the user's needs. The data sample will be downloaded to the user's local system in .json format.

Tutorials: There are four types of tutorials on the website:

  • Visualizaitions: This page gives example visualizations using the data from the database.
  • Page Categorization: This page discusses the use Latent Dirichlet Allocation. It gives examples of how this can be used with the database's data.
  • Text Sentiment: This page is a tutorial of how sentiment analysis can be used with the Youtube data and gives a line by line python tutorial on how to use sentiment analysis.
  • Review Analysis: This page is a tutorial of how to analyze reviews with the Yelp review data from the database. It also discusses how the neural network works and how to train it.

Help: This page provides users with an overview of the project. Providing addtional information for each of the website's subsections. Additionally, sample inference and download tutorials are provided.

GitHub: Provides a link to the GitHub hosting the source code for this project. Additional documentation can be found here as well as the various scripts written for this project.

Downloading Data

For users wishing to perform their own expirimentation using our dataset the following tutorial will explain the basic process of downloading a sample subset of our data. This example will show the process for the ABC News collection of data, however, the same steps can be followed for any other set of data. This dataset provides a good representation of the type of data we have collected, showing the change in the English language overtime.

The ABC (Australian Broadcasting Corporation) News contains over 1,000,000 news article headlines published over the course of 19 years. The dataset summarises a historical record of the most significant events that took place globally from the early 2000s to the early 2020s. While this dataset contains global events, however, the primary focus is on events that took place in Australia. The data was obtained by taking a sample of two hundred articles per day. These articles contain a good amount of international events, looking at the headlines, the most significant events can be singled out, showing how history has been shaped over the past two decades. Some examples of these events include elections, financial crises, or ecological disasters.
